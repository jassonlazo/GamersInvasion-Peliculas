use Dock menu:
- open plugins/pluginslist.xml
- replace gkplugins_youtube.swf to gkplugins_youtube.swf?dockmenu=true
timeslide option parameter (use with plugins_player.php):
- timeslidebgcolor : color of background timeslide, value is color code , Ex: 0xFF0000
- timeslidebgalpha : alpha of background timeslide, value(0~100)
- timeslidetimecolor : color of text time, value is color code , Ex: 0xFF0000
- timeslide : enable or disable timeslide, default is enable, Ex: timeslide=false (disable)
Box menu option parameter:
- hidelogo : hide logo youtube default is true, value(true/false)
- menu : show menu quality default is true, value(true/false)
- default : quality default is 360p, value(240p/360p/480p/720p/1080p)
- outcolor : color when mouse out, value is color code , Ex: 0xFF0000
- overcolor : color when mouse over, value is color code , Ex: 0xFF0000
- bgcolor : color of background list quality, value is color code , Ex: 0xFF0000
- bgalpha : alpha of background list quality, value(0~100)
- bgborder : color of border list quality, value is color code , Ex: 0xFF0000
Ex1: <plugins url="gkplugins_youtube.swf"/>
Ex2: <plugins url="gkplugins_youtube.swf?menu=false&hidelogo=false"/>
Ex3: <plugins url="gkplugins_youtube.swf?outcolor=0xFF0000&overcolor=0x00FF00&bgcolor=0x0000FF&bgalpha=30&bgborder=0xFFFFFF"/>
Ex4: <plugins url="gkplugins_youtube.swf?timeslidebgcolor=0xFF0000&timeslidebgalpha=70&timeslidetimecolor=0x00FF00"/>
